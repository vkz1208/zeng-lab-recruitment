## 当前项目状态摘要

当前代码库已经具备早期原型能力：官网/demo 学术网站页面、租户注册与邮箱验证、租户登录、租户内容存储、AI 草稿生成、租户后台、超级管理后台、Vercel 部署配置和本地开发服务器。

后续建设重点包括：正式平台官网、完整多租户管理、付费套餐、风格插件系统、域名绑定、动态页面、自动新闻/论文维护、微信小程序审核、访问统计、权限细分和生产环境安全能力。

## 使用原则

- `docs/SPEC.md` 只作为总入口、目录和高层摘要。
- 具体需求、约束和验收标准必须写入对应专题文档。
- 后续修改不得直接删除历史内容；失效内容需按 [00-文档治理](./spec/00-governance.md) 标注。
- 所有 SPEC 修改必须经过项目拥有者同意后才能生效。
