# 06. 安全与验收 SPEC

文档状态：已采纳/生效  
当前版本：v0.1-effective  
创建日期：2026-05-26
生效日期：2026-05-26

## 6.1 账号安全

- 密码必须加密存储。
- 会话 cookie 必须使用 HttpOnly。
- 生产环境必须使用强 session secret。
- 修改密码需要验证当前密码。
- 后续应支持密码重置、登录失败限制和双因素认证。

## 6.2 租户隔离

- 租户只能读取和修改自己的内容。
- 超级管理员可以管理所有租户。
- 租户内容、统计数据、审核队列必须按租户隔离。
- API 必须在服务端校验权限，不能依赖前端隐藏入口。

## 6.3 数据隐私

- 上传资料可能包含个人简历、学生信息、邮箱、论文草稿等敏感数据。
- AI 调用前应明确数据使用边界。
- 访问统计后台默认展示聚合数据。
- 原始 IP、用户代理等数据应限制访问和保留周期。

## 6.4 API Key 管理

- AI API key 只能存放在服务端环境变量或安全配置中。
- 不得将真实 key 写入仓库。
- 后续应支持平台级 key、租户级 key 或套餐级 key 策略。

## 6.5 注册与认证验收

- 用户可使用邮箱、密码、租户名称和 slug 注册。
- 重复邮箱和重复 slug 会被拒绝。
- 邮箱未验证时不能进入正式后台。
- 邮箱验证成功后可登录。
- 登录后可登出。
- 用户可修改密码。

## 6.6 初始化与 AI 生成验收

- 未初始化租户登录后进入初始化页面。
- 用户可上传资料并生成草稿。
- AI 配置正常时返回 AI 草稿。
- AI 未配置或失败时返回本地 fallback 草稿。
- 用户确认草稿后，租户标记为初始化完成。
- 初始化完成后进入租户后台。

## 6.7 多租户验收

- 租户 A 不能读取租户 B 内容。
- 租户 A 不能修改租户 B 内容。
- 独立域名和二级域名都应解析到正确租户。
- 禁用租户不能继续正常发布或访问后台。

## 6.8 后台验收

超级管理后台：

- 超级管理员可查看租户列表。
- 超级管理员可修改租户状态。
- 超级管理员可管理租户域名。
- 后续可管理套餐、功能开关和风格插件。

租户后台：

- 租户可编辑文本内容并保存。
- 租户可新增、删除、排序内容项。
- 租户可管理页面。
- 租户可切换风格。
- 租户可修改密码。
- 租户可查看访问统计。

## 6.9 自动维护与微信审核验收

- 系统可生成候选新闻或论文更新。
- 候选内容默认不自动发布。
- 用户可在微信小程序或后台审核。
- 用户同意后才发布到网站。
- 拒绝或修改意见需要记录。

## 6.10 视觉验收

- 桌面和移动端页面无明显布局错误。
- 文字不溢出父容器。
- 按钮、表单、卡片、导航无重叠。
- 中英文内容都可读。
- 风格切换后内容仍完整。
