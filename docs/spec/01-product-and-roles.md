# 01. 产品与角色 SPEC

文档状态：已采纳/生效  
当前版本：v0.1-effective  
创建日期：2026-05-26
生效日期：2026-05-26

## 1.1 产品目标

本项目是一个面向高校教授、PI、实验室和课题组的学术网页生成器与托管平台。平台通过资料上传、AI 内容生成、风格模板选择、可视化编辑和多租户托管，帮助学术用户快速生成并长期维护高质量个人或课题组网站。

## 1.2 目标用户

- 高校教授、研究员、PI。
- 实验室、课题组、研究中心。
- 需要展示论文、项目、团队、新闻、招生和联系信息的学术组织。
- 平台拥有者及运营人员。

## 1.3 平台拥有者需求

平台拥有者需要具备两类核心入口：

- 官网：承接用户注册、产品展示、套餐介绍、案例展示、登录入口和联系咨询。
- 平台管理后台：管理租户、套餐、服务项目、域名、风格模板、用户状态和后续运营功能。

## 1.4 租户用户价值

租户用户注册后，可以上传已有学术资料、选择视觉风格、使用 AI 生成网站草稿、预览并确认内容。确认后，系统为租户正式生成带管理后台的网站。租户可绑定独立域名，也可使用平台域名下的二级域名。

## 1.5 当前项目状态

已实现原型能力：

- 官网和 demo 学术网站页面：主页、团队、论文、研究方向、资源、新闻、加入、联系。
- 租户注册、邮箱验证、登录、登出、修改密码基础接口。
- 租户数据、用户数据、邮箱验证数据的本地 JSON 与 Vercel Blob 存储兼容机制。
- 租户内容独立存储机制。
- 租户初始化流程：上传资料文本、生成网站草稿、确认后写入租户内容。
- AI provider 抽象雏形：支持 provider、baseUrl、model、apiKey 等环境变量配置，当前默认 OpenAI 兼容接口。
- 租户后台原型：所见即所得内容编辑、保存生效、恢复默认。
- 超级管理后台原型：查看租户、修改租户状态、管理域名。
- Vercel 部署配置和本地开发服务器。

待建设能力：

- 正式平台官网与商业转化页面。
- 完整套餐与付费模式。
- 风格插件系统。
- 域名绑定自动化流程与 DNS 校验。
- 动态页面类型和页面管理系统。
- 更成熟的租户后台信息架构。
- 自动获取新闻、论文和其他学术动态的数据管道。
- 微信小程序审核、提醒和确认流程。
- 访问数据统计、IP 地域分布和后台可视化。
- 权限细分、协作者管理和审计日志。
- 面向生产环境的安全、监控、备份和合规能力。

## 1.6 用户与权限

平台拥有者 / 超级管理员：

- 查看、搜索、筛选所有租户。
- 启用、禁用、冻结或恢复租户。
- 管理租户套餐、服务项目和功能开关。
- 管理租户绑定域名和平台二级域名。
- 管理全局风格插件、模板资产和可用范围。
- 查看平台级运营数据和异常状态。
- 处理租户初始化失败、发布失败、域名异常等运营问题。

租户管理员：

- 完成账号注册、邮箱验证、登录、登出、密码修改。
- 上传资料并生成初始网站草稿。
- 预览并确认网站内容。
- 管理自己网站的页面、模块、文本、图片、论文、团队成员、新闻、项目等内容。
- 切换或配置可用风格。
- 管理域名绑定。
- 查看自己网站的基础访问数据。
- 审核自动抓取到的新闻、论文或其他候选更新。

租户协作者：

- 未来扩展角色，只能访问被授权的租户和内容范围。
- 可支持仅编辑新闻、论文、团队成员或仅查看访问数据。
- 可支持“提交修改但需租户管理员审核后发布”的工作流。

游客：

- 无需登录即可访问租户公开网站。
- 可查看主页、团队、论文、研究方向、新闻、项目、招生和联系方式等公开内容。
